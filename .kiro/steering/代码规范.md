# 代码规范

## 文件编码

- 所有文件必须使用 **UTF-8 编码**保存，禁止使用 GBK、GB2312、Latin-1 等其他编码
- 避免用 PowerShell 处理含中文的文件
- 创建或修改文件时，始终通过工具写入（fsWrite / strReplace），不要用系统文本编辑器另存为
- 写入 HTML / Python / JSON 等文件后，如有中文内容，用以下命令验证编码：
  ```bash
  python -c "open('path/to/file', encoding='utf-8').read(); print('OK')"
  ```
- 若发现文件含有损坏字节（`\ufffd` 替换符），立即用 Python 脚本修复并重新以 UTF-8 保存，不要忽略
