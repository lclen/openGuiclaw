# Product Overview

OpenGuiclaw 是一个基于 Qwen（通义千问）的智能桌面 AI 伙伴框架，支持 Web UI 和 CLI 两种交互方式。

## 核心能力

- **智能对话**：基于 OpenAI 兼容接口，支持 Function Calling 多轮工具链、联网搜索、多模态图片分析
- **视觉感知**：后台线程定时截屏，通过 Vision 模型分析屏幕状态，可主动发起对话（静默/正常/活泼三种模式）
- **记忆系统**：短期会话历史 + 长期记忆提取 + 向量语义检索（RAG）+ 知识图谱
- **自我进化**：每日自动回顾对话，提炼记忆，自动更新用户画像和 PERSONA.md 人设
- **GUI 自动化**：封装 pyautogui + mss，支持归一化坐标（0-1000）的视觉定位操作
- **计划执行**：将复杂目标分解为多步骤计划，支持自驾/确认/普通三种执行模式
- **插件与 MCP**：plugins/ 目录热加载，支持 MCP 协议网关，AI 可自主编写新插件
- **3D 虚拟形象**：Web UI 内置 Three.js + VRM 渲染，支持多模型切换

## 目标用户

个人开发者和技术用户，希望拥有一个可长期陪伴、持续进化、能操控桌面的 AI 助手。
